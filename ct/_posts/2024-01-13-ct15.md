## 문제

첫 번째 줄에 A, B, C (1 ≤ A, B, C ≤ 1012)이 공백을 사이에 두고 주어진다.

A+B+C의 값을 출력하라

## 코드

### 내 코드

```python
print(sum(list(map(int, input().split()))))
```

### 모범 코드

```python
print(eval(input().replace(*' +')))
```
- eval(문자열): 문자열 내 내용을 판별해서 알맞은 함수로 자동 변환

```python
print(eval("1+2")) 
# 출력값 3

c = eval("abs(-56)")
print(f'3. eval("abs(-56)") : {c}')
# abs(음수): 음수를 정수로 반화
# 출력값 56
```
