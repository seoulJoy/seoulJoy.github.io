# 어댑터 패턴

* toc
{:toc .large-only}

# 개요

![](/assets/img/cs/adapter.png)

- 클래스의 인터페이스를 사용자가 사용하고자 하는 다른 인터페이스로 변환하는 패턴
- 호환성이 없는 인터페이스를 갖는 클래스들을 호환시킴

# 특징
- 기존 코드 수정 없이 개조해서 필요 클래스 생성
- 소스코드 재사용 가능

# 예제

## 클래스 어댑터 패턴
- 상속을 이용한 어댑터 패턴
  - 다중 상속을 허용하는 프로그래밍 언어에서만 사용 가능

## 인스턴스 어댑터 패턴
- 위임을 이용한 어댑터 패턴
